# Pandas Question Solutions (Python v3.12)

This repository contains a number of questions related to Pandas Python and their solutions.
Students of Class 12th CBSE (INDIA) might be helped for their Secondary Boards Examination.

## About this repository
This repository consists of 20 questions and individual files for all of the questions.

- `questions/` -> Contains all the files of the 20 questions
- `questions/assets` -> Contains assets like `data.csv` to import data from the file. It is not necessary for you to create such file, I created it to organize the files.

- `LICENSE` -> License of the repository (ignore it)
- `requirements.txt` -> Contains all the modules/packages that you need to install to run the codes (you may ignore this too)

## How to use?
Look it the folder `questions`, click on it and you will see files like `1.py` and etc. Click on them and you can see the question on top of the file. Output has also been written in the python files itself.
